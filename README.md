# ZZU Mirror

> Ciallo! 欢迎使用郑州大学开源镜像站

这是一个高效的Linux系统镜像源配置工具，支持一键安装和配置各类常用软件源。

## ✨ 功能特性

- 🚀 **一键配置** - 自动检测系统并配置最适合的镜像源
- 📦 **丰富资源** - 支持Ubuntu、ArchLinux、MySQL、PostgreSQL等多种软件源
- ⚡ **高速下载** - 优化的镜像服务，提供稳定高速的下载体验
- 🛠️ **易于使用** - 简单的命令行界面，新手友好

## 🚀 快速开始

### 一键安装

```bash
bash <(curl -sSL https://mirrors.zzu.edu.cn/main.sh)
```

## 📚 支持的镜像源

- Ubuntu Releases
- Ubuntu Ports
- ArchLinux
- ArchLinux CN
- MySQL
- PostgreSQL
- Docker CE
- Qt
- CTAN

更多镜像源请查看 [docs/](docs/) 目录。
